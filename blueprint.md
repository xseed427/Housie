# **App Name**: Housie Empire

## Core Features:

- Role Selection: Allow users to select their role: owner, organizer, or player.
- Organizer Registration: Enable organizers to register by providing KYC details (name, phone, email, PAN, Aadhaar, bank details, city).
- Game Creation: Allow organizers to create new Housie games, setting parameters like title, ticket price and host cut.
- Game Room (Host): Provide a host view for managing the game, calling numbers, displaying the number board, and ending the game.
- Player Join: Enable players to join a game by entering a room code and paying for a ticket via Razorpay.
- Player Ticket Display: Display the player's generated ticket, highlight called numbers, and enable prize claims.
- Analytics Dashboard: Give the app owner access to overview analytics

## Style Guidelines:

- Primary color: Gold (#FFD700) to represent the potential earnings of users in the app.
- Background color: Dark gray (#1E1E1E) for a modern, sophisticated feel.
- Accent color: Pink (#E91E63) to draw attention to key interactive elements like called numbers and win claims.
- Body and headline font: 'Segoe UI' (sans-serif) as specified in the original code.
- Simple, geometric icons for key actions, like creating a new game or joining a room.
- Mobile-first, card-based layout, consistent with the container, card and other css settings used in the request.
- Subtle transitions and animations on button clicks and number calls.
